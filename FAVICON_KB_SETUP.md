# KB Favicon Setup Guide

## How to Create a KB Initials Favicon

To replace the favicon with a "KB" initials image, follow these steps:

### Option 1: Use an Online Favicon Generator (Easiest)

1. **Go to [Favicon.io](https://favicon.io/favicon-generator/)**
   - This is the easiest way to create a KB favicon

2. **Create your favicon:**
   - Text: `KB`
   - Background Color: Choose your primary color (blue)
   - Text Color: White
   - Font: Choose a bold font
   - Size: 32x32 or 64x64 pixels

3. **Download the favicon:**
   - Download the generated `favicon.png` file
   - Replace the existing `favicon.png` in `client/public/` folder

### Option 2: Create Manually

1. **Create a square image:**
   - Size: 32x32 pixels (or 64x64 for better quality)
   - Format: PNG with transparency
   - Background: Your primary color (blue)
   - Text: White "KB" in bold font

2. **Save as:**
   - File name: `favicon.png`
   - Location: `client/public/favicon.png`

### Option 3: Use Your Design Tool

1. **Using Photoshop, Figma, or Canva:**
   - Create a 32x32 or 64x64 pixel square
   - Add a blue background
   - Add white "KB" text in bold
   - Export as PNG
   - Save to `client/public/favicon.png`

### Quick Steps Summary

1. Create a 32x32 or 64x64 pixel square image
2. Add blue background
3. Add white "KB" text (bold)
4. Save as `favicon.png`
5. Place in `client/public/favicon.png`
6. Refresh browser to see the new favicon

### Recommended Tools

- **Favicon.io** - https://favicon.io/favicon-generator/ (Easiest)
- **Canva** - https://www.canva.com/ (Free design tool)
- **Figma** - https://www.figma.com/ (Professional design)
- **Photoshop** - If you have it

### Current Setup

The favicon is configured to use `/favicon.png` in the `client/public/` folder. Once you create and add your KB initials image, it will automatically appear in the browser tab.
